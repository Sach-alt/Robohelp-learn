‎APPENDIX A. Supported RADIUS RFCs

  • RFC 2865 (“Remote Authentication Dial In User Service (RADIUS)”) – supported.
  • RFC 2866 (“RADIUS Accounting”) – supported.
  • RFC 5090 (“RADIUS Extension for Digest Authentication”) – partially supported.
  • RFC 3579 (“RADIUS (Remote Authentication Dial In User Service) Support for Extensible Authentication Protocol (EAP)”) – partially supported.
  • RFC 3748 (“Extensible Authentication Protocol (EAP)”) – partially supported.
